# 🐹 Chomiczki

Strona zapowiadająca **Chomiczki** — prostą, śmieszną, online grę wyścigową dla dzieci 4–6 lat, w której chomiki ścigają się biegnąc w kółkach zamiast w pojazdach.

**Faza:** koncept. To materiał zapowiadający (w stylu Kickstartera) — gra nie jest jeszcze zakodowana. Kwoty i progi wsparcia na stronie są poglądowe.

🔗 **Strona:** https://marcindudekdev.github.io/gra-chomiczki/

## Zawartość
- `index.html` — strona zapowiadająca
- `art/` — koncept-arty (Grok Imagine) + teaser wideo
